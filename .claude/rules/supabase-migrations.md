## Supabase Migrations

### File Naming Convention

Migration files MUST follow: `YYYYMMDDHHmmss_short_description.sql`

- `YYYY` тАФ year (e.g., `2024`)
- `MM` тАФ month (01-12)
- `DD` тАФ day (01-31)
- `HH` тАФ hour, 24h format (00-23)
- `mm` тАФ minute (00-59)
- `ss` тАФ second (00-59)

Example: `20240906123045_create_profiles.sql`
